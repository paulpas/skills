# Test Automation Demo Skill

This skill demonstrates the automated skill relationship system.

## When to Use

- Testing the pre-commit hook
- Verifying skill automation
- Demonstrating auto-generated relationships

## Core Workflow

1. Add this skill to the repository
2. Stage it with git
3. Commit and watch the hook run
4. Hook validates, generates indexes, updates relationships
5. All files are auto-staged
